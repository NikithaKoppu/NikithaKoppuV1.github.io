In this internship, I spent my time fully integrated with the rest of the development team. I worked on over 66 stories and bugs and contributed to over 15,000 lines of code. While I was doing this, I was able to experiment with many different technologies like kafka stream apps, Java Spring Framework, Elasticsearch, docker containers, postgres databases, aws secrets, jenkins deployments etc. It was a great learning experience as the dev team was tight knit and collaborative effort was highly encouraged.

Project: Elasticsearch Events

One project in particular I was very excited to complete was the integration of elasticsearch for our audit events instead of using postgres. This project became more expansive as I dug into the effort because I also needed to get the infrastructure set up to handle the transition and the testing effort of the move. I worked closely with the devops team to address my concerns and what the proper solution should be for confidentiality and proper storage. This part of the process involved AWS, cloud elasticsearch clusters, and jenkins along with many iterations to verify the set up. The rest of the development effort was mine to work through which involved upgrading out es clients and clusters, juggling credentials, creating the right mappings, and getting kibana users set up. From start to finish, this was a fulfilling experience filled with challenges that I enjoyed tackling day to day.
